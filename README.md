# Spanish Tutor Pi
A headless Raspberry Pi Spanish tutor with a mobile-first web app.

## Setup Instructions
1. Flash Raspberry Pi OS Lite (64-bit) with username 'tutor'.
2. SSH into the Pi: ssh tutor@spanishpi.local
3. Clone this repo: git clone https://github.com/coryj6/spanish-tutor-pi.git
4. Run: bash scripts/install.sh
